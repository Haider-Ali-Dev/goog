## gog

Search on google from your terminal.

Supported Browsers:
- Chrome
- Firefox


Supported Operating Systems
- Linux


Unsupported or not tested operating systems
- Windows
- macOS


### Installation
#### Arch
```bash
$ git clone https://github.com/Haider-Ali-Dev/gog
```

```bash
$ makepkg -si
```

```bash
$ gog
```

#### Other Linux OS

Make sure you have Rust downloaded.


```bash
$ cargo install gog
```

```bash
$ gog
```
